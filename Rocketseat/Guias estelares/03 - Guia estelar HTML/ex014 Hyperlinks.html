Hyperlinks

a tag "a" é o que faz a Web ser tão divertida , ela nos permite nagevar entre um site e outro com uma enorme facilidade

Exemplo

Podemos permitir que o usuário faça uma navegação dentro do seu site usando a tag "a" e colocando o ID nos elementos e o mesmo nome dentro do href do link

Ou podemos navegar entre outros arquivos ou diretórios que temos na nossa máquina, usando a tag "a" e colocando ./ ou ../

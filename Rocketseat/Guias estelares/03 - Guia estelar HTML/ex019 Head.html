Como funciona o head

Meta - É importante para definirmos metadados do nosso site:

  • meta charset="UTF-8"
  • meta name="viewport" content="width-device-width, initial-scale="1.0"

Links no meta

  • link para favicon
  • link para css
  • Links para diferentes telas

SEO

  • meta name="author" content="Matheus França"
  • meta name="Description" content="........"
  • meta name="robots" content="follow, noindex, nofollow"
  • Temos o meta property para compartilhar coisas do facebook e o meta="name" para compartilhar do twitter

Vimos sobre Aninhamento, que é a forma em que o nosso código é organizado,basicamente mostrando quais são as tags filhas e as tags pais e também mantendo a indentação no nosso código.

Fluxo - É como se fosse uma lógica de programação, o nosso código ele é lido linha por linha de forma sequencial, isso pelo menos inicialmente no HTMl, nada que um IF e um ELSE não resolva isso.

Quebra de linha - Algumas tags tem por padrão a quebra de linha (Tag p por exemplo). Já a tag EM não tem essa quebra de linha. E também podemos usar a tag "br" - Break Roll para quebrar a linha

O que iremos aprender?

2º - Aprender a usar o Git e o GitHub.
3º - Desenvolvimento Web de fato! Estruturar documentos em HTML, personalizar com CSS e interagir com JavaScript.

5º - JavaScript em sua plenitude, utilizando recursos avançados como  HOF, Código assíncrono, Testes unitários, entre outros.
